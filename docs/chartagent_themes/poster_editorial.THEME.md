# Poster Editorial

Updated: 2026-04-07

## 1. Theme Identity

- Theme id: `poster_editorial`
- Short label: `Poster Editorial`
- One-line summary: Pinterest 계열의 포스터/매거진 spread 감각을 차트 surface에 옮긴 type-led editorial theme
- Artwork signature: type-led poster hierarchy, warm paper-like field, editorial spread contrast
- Best-fit contexts:
  - poster-style infographic
  - 큰 숫자와 짧은 문구가 중요한 chart
  - image 없이도 조형감이 필요한 table/chart
- Avoid when:
  - 기관형 dashboard
  - 실시간 방송형 긴장감이 필요한 경우

## 2. Source References

- `pinterest` infographic/editorial boards
- `notion`
- `claude`

Reference intent:

- Pinterest의 poster / spread / editorial collage 감각
- Notion의 warm neutral restraint
- Claude의 paper-like atmosphere

## 3. Visual Theme & Atmosphere

- Mood:
  - 포스터적
  - 설명적
  - 조형적
- Density:
  - `balanced`
- Contrast strategy:
  - warm paper base + strong title/value mass
- Surface language:
  - 차트가 component가 아니라 poster의 일부처럼 보임
- Motion expectation:
  - 낮은 움직임, composition reveal 중심

## 5. Typography Rules

- Title role:
  - stronger than in `editorial_outline`
- Label role:
  - compact but warm
- Value role:
  - poster stat처럼 큼직하게 허용
- Source role:
  - bottom-left footnote block

## 6. Geometry & Pattern

- card radius:
  - `26`
- chip radius:
  - `14`
- pattern mode:
  - `outline_plus_hatch`
- pattern spacing:
  - `18`
- guide lines:
  - 최소화

## 7. Family Defaults

- `bar_horizontal`
  - strong
- `comparison_table`
  - good
- `line`
  - good
- `donut`
  - okay
- `metric_wall`
  - good if type-led

## 8. Do

- 큰 타이포와 data를 조합해 poster 리듬을 만든다
- source는 footnote block처럼 둔다

## 9. Don't

- 일반 dashboard처럼 촘촘한 panel grid로 만들지 말 것
- 방송형 rail/tag를 강하게 넣지 말 것
