# Pinterest Composition Preset Seeds

Updated: 2026-04-07

목표:

- Pinterest editorial / infographic reference에서 얻은 공간 문법을
- `Auto Kairos`의 `creative_scene.composition` seed로 정리한다

## 1. `compare` -> `spread_gutter_compare`

Use when:

- 두 개 이상의 비교 대상을 magazine spread처럼 보여주고 싶을 때

Recommended composition:

- `canvas_strategy`: `mirrored_evidence`
- `anchor_topology`: `spread_gutter`
- `headline_zone`: `overline_chip`
- `module_zone`: `center_strip`
- `item_arrangement`: `paired_columns`
- `item_alignment`: `space_between`
- `negative_space_strategy`: `center_breathing_room`
- `text_surface`: `editorial_label`

Notes:

- 비교 대상은 좌우 균형보다 gutter 리듬이 중요
- 제목보다 각 영역 label과 수치 질량이 더 중요할 수 있음

## 2. `data` -> `poster_stat_cluster`

Use when:

- 숫자 몇 개와 짧은 설명을 poster처럼 묶고 싶을 때

Recommended composition:

- `canvas_strategy`: `evidence_cluster`
- `anchor_topology`: `type_led_mass`
- `headline_zone`: `none`
- `module_zone`: `center_cluster`
- `item_arrangement`: `metric_cluster`
- `item_alignment`: `start`
- `negative_space_strategy`: `poster_margin`
- `text_surface`: `poster_stat`

Notes:

- 큰 title이 없어도 됨
- 숫자와 label이 화면 구조를 이끄는 것이 핵심

## 3. `quote_pull` -> `image_quote_collage`

Use when:

- 인용문, 캡션, 작은 이미지가 collage처럼 붙는 장면

Recommended composition:

- `canvas_strategy`: `voice_frame`
- `anchor_topology`: `single_quote_mass`
- `headline_zone`: `none`
- `module_zone`: `quote_frame`
- `item_arrangement`: `attribution_tail`
- `item_alignment`: `start`
- `negative_space_strategy`: `quote_breathing_room`
- `text_surface`: `footnote_block`

Notes:

- quote 자체가 hero다
- source와 attribution은 작지만 구조적으로 중요하다

## 4. Recommended new text roles

- `poster_stat`
- `section_kicker`
- `data_caption`
- `footnote_block`
- `spread_label`

- `spread_gutter`
